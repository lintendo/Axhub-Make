# Ant Design 组件展示页面

## 1. 简要概述
Ant Design 组件展示页面是一个综合性的演示页面，用于展示如何在 Axhub 页面中集成和使用 Ant Design v6 组件库。该页面包含了常用的 UI 组件（按钮、输入框、表格等），演示了完整的页面开发规范和最佳实践。

## 2. 功能说明
按照页面或元素展开说明所有功能，需要将每个模块都详细展开，不要遗漏任何部分。

### 2.1 页面头部
- **组成结构**：页面标题和说明文字
- **内容类型**：可配置的标题文本
- **布局和样式**：
  - 显示在页面顶部
  - 使用较大字号
  - 包含简短的说明文字
- **交互行为**：无交互
- **状态变化**：无状态变化

### 2.2 按钮组件卡片
- **组成结构**：包含多种类型的按钮展示
- **内容类型**：
  - 主要按钮（Primary）
  - 默认按钮（Default）
  - 虚线按钮（Dashed）
  - 文本按钮（Text）
  - 链接按钮（Link）
  - 危险按钮（Danger）
  - 加载中按钮（Loading）
  - 禁用按钮（Disabled）
- **布局和样式**：
  - 使用 Ant Design Card 组件包裹
  - 按钮水平排列，有适当间距
  - 使用 Ant Design 默认主题样式
- **交互行为**：
  - 点击主要按钮：触发 `onButtonClick` 事件，传递时间戳
  - 其他按钮：展示不同的视觉效果
- **状态变化**：无状态变化

### 2.3 输入组件卡片
- **组成结构**：包含多种输入控件
- **内容类型**：
  - 文本输入框（Input）
  - 搜索框（Search）
  - 日期选择器（DatePicker）
  - 开关（Switch）
- **布局和样式**：
  - 使用 Ant Design Card 组件包裹
  - 控件垂直排列，有适当间距
  - 每个控件有标签说明
- **交互行为**：
  - 支持文本输入
  - 支持日期选择
  - 支持开关切换
  - 可与外部系统集成
- **状态变化**：输入值变化（内部状态）

### 2.4 数据表格卡片
- **组成结构**：
  - 刷新按钮
  - 数据表格
  - 选中行信息展示区域
- **内容类型**：
  - 表格列：ID、姓名、年龄、地址、标签
  - 表格数据：来自 `tableData` 数据源
  - 分页控件
- **布局和样式**：
  - 使用 Ant Design Card 和 Table 组件
  - 表格支持分页，默认每页 5 条
  - 选中行有背景色高亮
  - 标签使用 Ant Design Tag 组件
- **交互行为**：
  - 点击刷新按钮：显示加载动画（1秒），可通过 `refreshData` 动作触发
  - 点击表格行：选中该行，触发 `onTableRowClick` 事件，传递行数据
  - 在表格下方显示选中行的详细信息
- **状态变化**：
  - `selectedRow` 变量：当前选中的行数据
  - `loading` 状态：表格加载状态

## 3. Axure API 说明
> 如果没有使用 Axure API，则不需要编写此部分。

### 事件列表
| 事件名 | 描述 | 触发时机 | 参数 |
|--------|------|----------|------|
| `onButtonClick` | 按钮点击时触发 | 点击主要按钮时 | `{ timestamp: number }` - 点击时的时间戳 |
| `onTableRowClick` | 表格行点击时触发 | 点击表格任意行时 | `{ row: object }` - 被点击的行数据 |

### 动作列表
| 动作名 | 描述 | 参数 |
|--------|------|------|
| `refreshData` | 刷新数据 | 无 |

### 变量列表
| 变量名 | 类型 | 描述 |
|--------|------|------|
| `selectedRow` | `object \| null` | 当前选中的行数据，未选中时为 null |

### 配置项列表
| 配置项ID | 类型 | 显示名称 | 说明 | 默认值 |
|----------|------|----------|------|--------|
| `title` | `input` | 页面标题 | 显示在页面顶部的标题 | `'Ant Design 组件展示'` |

### 数据项列表
| 数据项名称 | 描述 | 数据结构 |
|-----------|------|----------|
| `tableData` | 表格数据 | `Array<TableRow>` |

**TableRow 字段说明：**
- `id` (number)：用户 ID，唯一标识
- `name` (string)：姓名
- `age` (number)：年龄
- `address` (string)：地址
- `tags` (string[])：标签数组，用于分类

**数据示例：**
```json
{
  "tableData": [
    { "id": 1, "name": "张三", "age": 32, "address": "北京市朝阳区", "tags": ["开发", "设计"] },
    { "id": 2, "name": "李四", "age": 28, "address": "上海市浦东新区", "tags": ["产品"] }
  ]
}
```
