# 按钮组件

## 📋 业务与功能

### 1.1 核心目标
> 提供基础交互按钮元素，演示计数、消息展示与配置项的联动能力，适用于轻量交互场景。

该元素强调简单清晰的交互反馈与可配置性。

### 1.2 功能清单
> 列出所有功能模块及其优先级

- **标题区域**：展示组件标题
- **控制区域**：主按钮（计数 + 图标）与重置按钮
- **消息区域**：展示可配置消息文本
- **数据展示区域**：展示 data1 数据列表

### 1.3 交互要点
> 关键的交互触发点、反馈机制、状态变化

- 点击主按钮：计数 +1，触发 `onClick` 与 `onCountChange` 事件
- 点击重置按钮：计数回到初始值，触发 `onCountChange` 事件
- 动作 `increment`/`reset` 可外部调用
- 动作 `setMessage` 可动态更新消息内容

## ⚙️ Axure API 说明

### 5.1 事件列表（eventList）
> 组件对外暴露的事件

| 事件名称 | Payload 类型 | 触发时机 | 说明 |
|---------|-------------|---------|------|
| `onClick` | `{ count: number }` | 点击主按钮时 | 返回当前计数 |
| `onCountChange` | `{ count: number, action: string }` | 计数变化时 | 返回新计数与动作类型 |

### 5.2 动作列表（actionList）
> 组件可被调用的动作

| 动作名称 | Params 类型 | 参数说明 | 功能描述 |
|---------|------------|---------|---------|
| `increment` | `void` | 无 | 计数 +1 |
| `reset` | `void` | 无 | 重置计数 |
| `setMessage` | `{ message: string }` | 消息内容 | 更新消息文本 |

### 5.3 变量列表（varList）
> 组件内部状态变量

| 变量名称 | 类型 | 默认值 | 说明 |
|---------|-----|-------|------|
| `count` | `number` | `0` | 当前计数值 |
| `message` | `string` | `这是一个 React 元素示例` | 消息内容 |

**命名规范**：沿用组件现有命名（camelCase）

### 5.4 配置项列表（configList）
> 组件配置项

| 配置项名称 | 类型 | 默认值 | 说明 |
|----------|-----|-------|------|
| `title` | `string` | `React 示例` | 组件标题 |
| `buttonText` | `string` | `点击我` | 主按钮文本 |
| `primaryColor` | `string` | `#1890ff` | 主色调 |
| `initialCount` | `number` | `0` | 初始计数值 |
| `message` | `string` | `这是一个 React 元素示例` | 消息内容 |

### 5.5 数据项列表（dataList）
> 组件数据结构定义

**数据结构**：
```typescript
{
  data1: Array<{
    name: string;  // 名称
    value: any;    // 值
  }>;
```
