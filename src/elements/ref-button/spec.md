# 按钮组件

## 1. 简要概述
按钮组件是一个基础交互元素，用于展示 Axhub 组件开发规范和最佳实践。该组件提供了点击交互、状态管理、事件触发等核心功能，适用于需要用户操作触发的场景。

## 2. 功能说明
按照页面或元素展开说明所有功能，需要将每个模块都详细展开，不要遗漏任何部分。

### 2.1 标题区域
- **组成结构**：文本标题
- **内容类型**：可配置的标题文本
- **布局和样式**：
  - 显示在组件顶部
  - 颜色可通过 `primaryColor` 配置项自定义
  - 使用较大字号，突出显示
- **交互行为**：无交互
- **状态变化**：无状态变化

### 2.2 控制区域
- **组成结构**：包含主按钮和重置按钮
- **内容类型**：
  - 主按钮：显示可配置的按钮文本和当前计数值
  - 重置按钮：固定文本"重置"
- **布局和样式**：
  - 两个按钮水平排列
  - 主按钮使用主色调背景
  - 重置按钮使用灰色背景
  - 采用现代化的圆角设计
- **交互行为**：
  - 主按钮点击：计数器加 1，触发 `onClick` 和 `onCountChange` 事件
  - 重置按钮点击：计数器重置为初始值，触发 `onCountChange` 事件
  - 按钮有悬停和点击的视觉反馈
- **状态变化**：
  - `count` 变量随点击和重置操作变化

### 2.3 消息区域
- **组成结构**：文本显示区域
- **内容类型**：可配置的消息文本，可通过动作动态更新
- **布局和样式**：
  - 显示在按钮下方
  - 使用常规字号和颜色
- **交互行为**：无直接交互
- **状态变化**：
  - `message` 变量可通过 `setMessage` 动作更新

### 2.4 数据展示区域
- **组成结构**：数据列表展示
- **内容类型**：来自 `data1` 数据源的键值对列表
- **布局和样式**：
  - 以列表形式展示
  - 每项显示名称和值
- **交互行为**：无交互
- **状态变化**：数据源变化时自动更新

## 3. Axure API 说明
> 如果没有使用 Axure API，则不需要编写此部分。

### 事件列表
| 事件名 | 描述 | 触发时机 | 参数 |
|--------|------|----------|------|
| `onClick` | 点击按钮时触发 | 点击主按钮时 | `{ count: number }` - 当前计数值 |
| `onCountChange` | 计数器改变时触发 | 计数器值发生变化时 | `{ count: number, action: string }` - 新计数值和动作类型 |

### 动作列表
| 动作名 | 描述 | 参数 |
|--------|------|------|
| `increment` | 计数器加1 | 无 |
| `reset` | 重置计数器 | 无 |
| `setMessage` | 设置消息内容 | `{ message: string }` - 新的消息内容 |

### 变量列表
| 变量名 | 类型 | 描述 |
|--------|------|------|
| `count` | `number` | 当前计数值 |
| `message` | `string` | 消息内容 |

### 配置项列表
| 配置项ID | 类型 | 显示名称 | 说明 | 默认值 |
|----------|------|----------|------|--------|
| `title` | `input` | 组件标题 | 组件顶部显示的标题文本 | `'React 组件示例'` |
| `buttonText` | `input` | 按钮文本 | 主按钮显示的文本内容 | `'点击我'` |
| `primaryColor` | `colorPicker` | 主色调 | 组件的主题颜色 | `'#1890ff'` |
| `initialCount` | `inputNumber` | 初始计数值 | 计数器的初始值 | `0` |
| `message` | `input` | 消息内容 | 组件显示的消息文本 | `'这是一个 React 组件示例'` |

### 数据项列表
| 数据项名称 | 描述 | 数据结构 |
|-----------|------|----------|
| `data1` | 基础数据列表 | `Array<{ name: string, value: any }>` |

**data1 字段说明：**
- `name` (string)：名称
- `value` (any)：值
