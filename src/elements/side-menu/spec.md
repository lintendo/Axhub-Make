# 侧边菜单

## 📋 业务与功能

### 1.1 核心目标
> 提供后台类页面的侧边导航组件，支持多级菜单、图标展示与折叠收起，提升导航效率。

该元素强调结构清晰、信息密度适中，适配后台场景。

### 1.2 功能清单
> 列出所有功能模块及其优先级

- **顶部标题区域**：显示应用或系统名称
- **菜单项区域**：支持一级/二级菜单项、图标与禁用状态
- **折叠按钮**：控制侧边栏折叠/展开

### 1.3 交互要点
> 关键的交互触发点、反馈机制、状态变化

- 点击菜单项：选中菜单项并触发 `onMenuSelect`
- 点击折叠按钮：切换折叠状态并触发 `onCollapseChange`
- 子菜单展开/收起：点击父级菜单项控制
- 悬停菜单项：高亮反馈

## 📊 内容规划

### 2.1 信息架构
> 模块划分、信息层级、内容组织方式

```
侧边菜单
├── 顶部标题
```

### 2.2 数据来源
> 数据源优先级：用户提供数据 > 项目数据表 > 生成示例数据

- **数据类型**：菜单项列表（`items`）
- **数据源**：用户提供（props）/ 内置默认菜单
- **关键字段**：
  - `key`: 唯一标识
  - `label`: 显示文本
  - `icon`: 图标名称（dashboard / shop / user / setting）
  - `disabled`: 是否禁用
  - `children`: 子菜单数组

### 2.3 内容示例
> （可选）重要的示例内容、文案语气、术语规范

**默认菜单示例**：

## 🎨 布局与结构

### 3.1 整体布局
> 布局模式（单栏/双栏/网格/自由）、模块尺寸、比例约束

- **布局模式**：垂直侧边栏
- **容器宽度**：固定
- **关键尺寸**：
  - 展开宽度：`240px`
  - 折叠宽度：`64px`
  - 顶部标题区高度：`48px`

### 3.2 响应式适配
> （如适用）断点、适配策略

- **桌面端（≥1200px）**：默认展开显示
- **平板端（768-1199px）**：建议默认折叠或在主区保留更大空间
- **移动端（<768px）**：建议折叠或使用抽屉式替代

## ⚙️ Axure API 说明
> 本组件未接入 Axure API，以下为 React Props 约定

### 5.1 事件列表（eventList）
> 组件对外暴露的事件

| 事件名称 | Payload 类型 | 触发时机 | 说明 |
|---------|-------------|---------|------|
| `onMenuSelect` | `string` | 点击菜单项时 | 返回菜单项 key |
| `onCollapseChange` | `boolean` | 折叠状态变化时 | 返回折叠状态 |

### 5.2 动作列表（actionList）
> 组件可被调用的动作

| 动作名称 | Params 类型 | 参数说明 | 功能描述 |
|---------|------------|---------|---------|
| - | - | - | 无 |

### 5.3 变量列表（varList）
> 组件内部状态变量

| 变量名称 | 类型 | 默认值 | 说明 |
|---------|-----|-------|------|
| - | - | - | 无 |

**命名规范**：沿用组件现有命名（camelCase）

### 5.4 配置项列表（configList）
> 组件配置项

| 配置项名称 | 类型 | 默认值 | 说明 |
|----------|-----|-------|------|
| `title` | `string` | `Axhub` | 顶部标题文本 |
| `width` | `number` | `240` | 展开宽度 |
| `collapsible` | `boolean` | `true` | 是否显示折叠按钮 |
| `defaultCollapsed` | `boolean` | `false` | 初始是否折叠 |
| `defaultSelectedKey` | `string` | `dashboard` | 初始选中项 |
| `items` | `MenuItem[]` | `[]` | 菜单项列表 |

### 5.5 数据项列表（dataList）
> 组件数据结构定义

**数据结构**：
```typescript
{
  items: Array<{
    key: string;        // 菜单项唯一标识
    label: string;      // 显示文本
    icon?: string;      // 图标名称
    disabled?: boolean; // 是否禁用
    children?: Array<{  // 子菜单项
      key: string;
      label: string;
      icon?: string;
      disabled?: boolean;
    }>;
  }>;
}
```
