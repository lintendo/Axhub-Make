# 侧边菜单

## 1. 简要概述
侧边菜单是一个基于 Ant Design 的导航组件，用于在页面中快速搭建后台类布局的侧边导航。支持菜单项配置、折叠展开、图标显示等功能。

## 2. 功能说明
按照页面或元素展开说明所有功能，需要将每个模块都详细展开，不要遗漏任何部分。

### 2.1 顶部标题区域
- **组成结构**：标题文本
- **内容类型**：可配置的标题文本
- **布局和样式**：
  - 显示在菜单顶部
  - 居中对齐
  - 使用较大字号和加粗字体
- **交互行为**：无交互
- **状态变化**：无状态变化

### 2.2 菜单项区域
- **组成结构**：多级菜单列表，支持父子层级
- **内容类型**：
  - 菜单项标签文本
  - 菜单项图标（可选）
  - 子菜单项（可选）
- **布局和样式**：
  - 垂直排列
  - 选中项有高亮背景色
  - 支持图标和文本组合显示
  - 子菜单有缩进效果
- **交互行为**：
  - 点击菜单项：选中该项，触发 `onMenuSelect` 回调，传递菜单项 key
  - 点击父菜单项：展开/收起子菜单
  - 悬停时有视觉反馈
- **状态变化**：
  - 当前选中的菜单项 key 变化
  - 子菜单展开/收起状态变化

### 2.3 折叠按钮区域
- **组成结构**：折叠/展开控制按钮
- **内容类型**：图标按钮
- **布局和样式**：
  - 显示在菜单底部或固定位置
  - 使用箭头图标指示当前状态
- **交互行为**：
  - 点击按钮：切换菜单折叠/展开状态
  - 触发 `onCollapseChange` 回调，传递折叠状态
- **状态变化**：
  - 菜单宽度在展开宽度和折叠宽度之间切换
  - 菜单项文本在折叠时隐藏

## 3. Props 配置

### 配置项列表
| 配置项 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| `title` | `string` | 顶部标题文本 | `'Axhub'` |
| `width` | `number` | 展开宽度（像素） | `240` |
| `collapsible` | `boolean` | 是否显示折叠按钮 | `true` |
| `defaultCollapsed` | `boolean` | 初始是否折叠 | `false` |
| `defaultSelectedKey` | `string` | 初始选中项 key | `'dashboard'` |
| `items` | `array` | 菜单项数组 | `[]` |

### 菜单项数据结构
```typescript
interface MenuItem {
  key: string;           // 菜单项唯一标识
  label: string;         // 显示文本
  icon?: string;         // 图标名称（支持：dashboard / shop / user / setting）
  children?: MenuItem[]; // 子菜单项
}
```

### 回调函数
| 回调名 | 描述 | 参数 |
|--------|------|------|
| `onMenuSelect` | 点击菜单项时触发 | `(key: string)` - 被点击的菜单项 key |
| `onCollapseChange` | 折叠状态变化时触发 | `(collapsed: boolean)` - 当前折叠状态 |
